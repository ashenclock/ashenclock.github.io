Privacy Notice (GDPR)

Last updated: 2026-05-05

This website is a static portfolio. In line with GDPR transparency principles (Articles 12–14), this notice explains what data processing may occur.

Technical processing by infrastructure providers

The site is hosted on GitHub Pages. Standard server and network logs (for security and delivery) may be processed by hosting/CDN providers when pages are requested.

Some static assets are loaded via public CDNs. Request metadata (such as IP address and user-agent) may be processed by those providers to deliver content.

Legal basis

For strictly necessary technical delivery, processing is based on legitimate interest in secure and reliable website operation.

Data subject rights
