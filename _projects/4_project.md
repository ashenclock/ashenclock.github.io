RoboExhibit is an AI-driven virtual museum guide that handles multi-step user queries.

- **Stack:** Unity 3D, C#, LLM integration, SPARQL, ontology-based knowledge base
- Implemented a neuro-symbolic architecture where an LLM produces structured SPARQL queries.
- Combined generative flexibility with symbolic grounding for more reliable answers.

Repository: [ROBOEXHIBIT](https://github.com/ashenclock/ROBOEXHIBIT)
